LABORATORIO NUMERO DOS

En este laboratorio se trabajó con la gestión y manipulación de una base de datos de una escuela utilizando diversas sentencias SQL para realizar las siguientes tareas:

CONSULTAS

    1. Inserción de datos: Con el comando INSERT INTO se añadieron nuevos registros a las tablas de la base de datos.

    2. Creación de tablas: Mediante CREATE TABLE, se diseñaron estructuras para almacenar datos, definiendo campos y tipos de datos.

    3. Actualización de registros: Se realizaron modificaciones en los datos existentes utilizando UPDATE.

    4. Eliminación de registros: Con DELETE, se eliminaron registros específicos de las tablas.

    5. Cálculos y agregaciones: Se empleó COUNT para contar registros y obtener estadísticas de los datos.

    6. Validación de existencia: Usando EXISTS, se verificaron condiciones específicas antes de realizar operaciones.

    7. Ordenamiento: Se organizaron los datos según criterios específicos mediante ORDER BY.

El objetivo principal fue aplicar estas consultas en un contexto práctico, mejorando el entendimiento de cómo gestionar bases de datos en un entorno educativo, asegurando una manipulación de datos precisa y eficiente
