LABORATORIO NUMERO TRES

En este laboratorio se desarrollaron consultas SQL en una base de datos enfocada en la gestión de recursos humanos de un colegio. Se aplicaron diversas operaciones para manipular y analizar datos, como:

CONSULTAS

    1. Búsqueda y filtrado de información: Utilizando SELECT y SELECT DISTINCT para obtener datos específicos y evitar duplicados, así como combinaciones de condiciones mediante AND y OR para mayor precisión en las consultas.

    2. Ordenamiento y limitación de resultados: Empleo de ORDER BY para organizar los datos en un orden específico y LIMIT para restringir la cantidad de resultados obtenidos.

    3. Eliminación de registros: Implementación del comando DELETE para eliminar datos según ciertas condiciones.

    4. Análisis de datos: Uso de funciones como COUNT para contar registros, SUM para realizar sumatorias, y la verificación de condiciones con EXISTS.

    5. Manejo de valores nulos: Identificación y gestión de valores NULL para asegurar la integridad de los datos.

El laboratorio permitió reforzar habilidades en la construcción de consultas complejas y la manipulación de datos dentro de una base de datos estructurada.
